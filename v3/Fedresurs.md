# Данные о банкротствах и существенных фактах
Метод позволяет получить информацию с сайта федресурса о банкротствах и существенных фактах.

## Параметры запроса

Для получения информации необходим запрос `publications`.

Данный запрос выполняется с наличием обязательных фильтров:
* `(first: 5)` - число задает количество выданных ответов от нового к старым.
* `(filter:{type:BANKRUPTCY})` - для банкротств
* `(filter:{type:SIGNIFICANT_FACT})` - для существенных фактов


## Пример запроса существенных фактов

``` graphql
{
  company(ogrn: "1025900893886"){
    publications{messages(first:2 filter:{type:SIGNIFICANT_FACT}){
        count{
          total
          local
        }
        data{
          id
          number
          externalId
          url
          publicationDate
          isAnnuled
          isRefuted
          isLocked
          title
          text
          factType
          publisher{
            name
            entity{
              __typename
            }
            publisherType
          }
          notary{
            name
            title
          }
          bankruptcy{
            caseNumber
          }
          objectId
          info{
            __typename
          }
        }
      }
    }
  }
}
```
